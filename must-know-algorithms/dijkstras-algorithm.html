Dijkstra's Algorithm

Single-source shortest paths on a weighted graph with non-negative edges — greedy relaxation with a min-heap.

📅 April 5, 2026

Problem

Given a weighted undirected graph and a source vertex src, compute the shortest path distance from src to every other vertex. The graph has no negative edge weights (standard Dijkstra requirement).

Example

src = 0, adjacency list adj (each entry is [neighbor, weight]):

adj[0] = [[1, 4], [2, 8]] adj[1] = [[0, 4], [2, 3], [4, 6]] adj[2] = [[0, 8], [1, 3], [3, 2]] adj[3] = [[2, 2], [4, 10]] adj[4] = [[1, 6], [3, 10]]

Output: [0, 4, 7, 9, 10]

Shortest paths from 0

  • 0 → 0: distance 0 (source).
  • 0 → 1: direct edge 4.
  • 0 → 2: 0 → 1 → 2 gives 4 + 3 = 7 (better than direct 8).
  • 0 → 3: 0 → 1 → 2 → 3 gives 4 + 3 + 2 = 9.
  • 0 → 4: 0 → 1 → 4 gives 4 + 6 = 10.

Code

Min-heap stores (distance, vertex); relax neighbors when a shorter path is found. Copy the full snippet from the toolbar.

import java.util.*;

class Solution {

    // adj.get(u) is list of [v, weight]; graph is undirected, non-negative weights
    public int[] dijkstra(int V, ArrayList<ArrayList<ArrayList<Integer>>> adj, int src) {

        int[] dist = new int[V];
        Arrays.fill(dist, Integer.MAX_VALUE);
        dist[src] = 0;

        PriorityQueue<int[]> pq = new PriorityQueue<>((a, b) -> Integer.compare(a[0], b[0]));
        pq.offer(new int[]{0, src});

        while (!pq.isEmpty()) {
            int[] cur = pq.poll();
            int d = cur[0];
            int u = cur[1];

            if (d > dist[u]) {
                continue;
            }

            for (ArrayList<Integer> edge : adj.get(u)) {
                int v = edge.get(0);
                int w = edge.get(1);
                if (dist[u] != Integer.MAX_VALUE && dist[u] + w < dist[v]) {
                    dist[v] = dist[u] + w;
                    pq.offer(new int[]{dist[v], v});
                }
            }
        }

        return dist;
    }
}
