Detect Cycle in Graph using Disjoint Set Union (DSU)

Use union–find while adding edges: if two endpoints are already in the same set, adding this edge closes a cycle.

📅 April 5, 2026

Context

For an undirected graph, we already know how to detect a cycle with BFS or DFS (e.g. visiting a node that is not the parent). DSU (Disjoint Set Union) is another clean way to solve the same problem: process edges and merge components; a union of two vertices that are already connected reveals a cycle.
