54. Spiral Matrix

Return all elements of an m × n matrix in spiral order — clockwise from the outside layer inward. Maintain four boundaries (top, bottom, left, right), peel one layer per loop: go right along the top row, down the right column, left along the bottom row (if a row remains), then up the left column (if a column remains).

Microsoft · Medium — 14 April 2026

Summary

Hint: After each edge walk, shrink the corresponding boundary (top++, right--, bottom--, left++). The two inner if checks matter when the matrix collapses to a single row or single column — without them you can revisit the same cells or read an empty strip.

Answer

✅ Layer boundaries — four pointers

✅ Solution.java — boundary spiral 
import java.util.*;

class Solution {
    public List<Integer> spiralOrder(int[][] matrix) {

        List<Integer> result = new ArrayList<>();

        int m = matrix.length;
        int n = matrix[0].length;

        int top = 0;
        int bottom = m - 1;
        int left = 0;
        int right = n - 1;

        while (top <= bottom && left <= right) {

            // left -> right
            for (int i = left; i <= right; i++) {
                result.add(matrix[top][i]);
            }
            top++;

            // top -> bottom
            for (int i = top; i <= bottom; i++) {
                result.add(matrix[i][right]);
            }
            right--;

            // right -> left (only if valid row exists)
            if (top <= bottom) {
                for (int i = right; i >= left; i--) {
                    result.add(matrix[bottom][i]);
                }
                bottom--;
            }

            // bottom -> top (only if valid column exists)
            if (left <= right) {
                for (int i = bottom; i >= top; i--) {
                    result.add(matrix[i][left]);
                }
                left++;
            }
        }

        return result;
    }
}
Why the two ifs. After the first two edges, you might have consumed the entire matrix (e.g. one row tall). The bottom row walk only runs if top ≤ bottom; the left column walk only if left ≤ right. Example: [[1,2,3]] — after going right, top > bottom, so skip the bottom-row pass and avoid adding 2 again.

Complexity

  • Time: O(m · n) — each cell is visited exactly once.
  • Space: O(m · n) for the output list (excluding the input matrix); only O(1) extra variables beyond that.
