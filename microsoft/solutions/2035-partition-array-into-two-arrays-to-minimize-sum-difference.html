2035. Partition Array Into Two Arrays to Minimize Sum Difference

You are given an integer array nums of length 2 * n. Split nums into two arrays nums1 and nums2 of length n each such that nums1 and nums2 together contain all elements of nums (each element appears exactly once). Return the minimum possible value of |sum(nums1) - sum(nums2)|.

Microsoft · Hard

Summary

This is a constrained partition problem: choose exactly half the elements for one side so the two subset sums are as close as possible. Brute force over all C(2n, n) splits is infeasible for typical constraints; the usual interview direction involves structured search (for example meet-in-the-middle on the two halves of the array) or DP over counts and sums.
