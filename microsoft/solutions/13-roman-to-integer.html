13. Roman to Integer

Convert a Roman numeral string to its integer value by resolving subtractive pairs first and handling remaining single symbols.

Microsoft · Easy

Summary

Hint: Scan from left to right. First check if current and next characters form one of the 6 subtractive pairs (IV, IX, XL, XC, CD, CM). If yes, add that value and jump by 2; otherwise add single-symbol value and move by 1.

Answers

Naive approach

Solution.java — naive 
class Solution {
    public int romanToInt(String s) {
        int i = 0;
        int total = 0;

        while (i < s.length()) {

            // check 2-character cases first
            if (i + 1 < s.length()) {
                String pair = s.substring(i, i + 2);

                if (pair.equals("IV")) { total += 4; i += 2; continue; }
                if (pair.equals("IX")) { total += 9; i += 2; continue; }
                if (pair.equals("XL")) { total += 40; i += 2; continue; }
                if (pair.equals("XC")) { total += 90; i += 2; continue; }
                if (pair.equals("CD")) { total += 400; i += 2; continue; }
                if (pair.equals("CM")) { total += 900; i += 2; continue; }
            }

            // otherwise single character
            total += getValue(s.charAt(i));
            i++;
        }

        return total;
    }

    private int getValue(char c) {
        if (c == 'I') return 1;
        if (c == 'V') return 5;
        if (c == 'X') return 10;
        if (c == 'L') return 50;
        if (c == 'C') return 100;
        if (c == 'D') return 500;
        return 1000;
    }
}
Direct pair matching. Time O(n), space O(1).

✅ Optimized approach

✅ Solution.java — optimized 
class Solution {
    public int romanToInt(String s) {
        int total = 0;

        for (int i = 0; i < s.length(); i++) {
            int curr = getValue(s.charAt(i));

            // check next character
            if (i < s.length() - 1 && curr < getValue(s.charAt(i + 1))) {
                total -= curr;  // subtract case
            } else {
                total += curr;  // normal case
            }
        }

        return total;
    }

    private int getValue(char c) {
        if (c == 'I') return 1;
        if (c == 'V') return 5;
        if (c == 'X') return 10;
        if (c == 'L') return 50;
        if (c == 'C') return 100;
        if (c == 'D') return 500;
        return 1000; // 'M'
    }
}
Subtractive rule in one pass. Time O(n), space O(1).
