210. Course Schedule II

Same graph as 207. Course Schedule — but instead of returning a boolean, return one valid topological order. Same Kahn's algorithm: build indegree, BFS the zero-indegree nodes, but this time record each node we poll into a result array.

Hint

Same as 207: BFS with indegree — start from nodes with 0 indegree and reduce dependencies. The only delta: keep an order[] and write each polled u into it. If you fill all numCourses slots, return order; otherwise (cycle blocked progress) return new int[0].

// Kahn's algo — record the order

import java.util.*;

class Solution {
    public int[] findOrder(int numCourses, int[][] prerequisites) {

        // Step 1: adjacency list
        ArrayList<ArrayList<Integer>> adj = new ArrayList<>();
        for (int i = 0; i < numCourses; i++) {
            adj.add(new ArrayList<>());
        }

        // Step 2: indegree
        int[] indegree = new int[numCourses];

        for (int[] pre : prerequisites) {
            int course = pre[0];
            int prereq = pre[1];

            adj.get(prereq).add(course); // edge: prereq -> course
            indegree[course]++;
        }

        // Step 3: queue (indegree 0)
        Queue<Integer> q = new LinkedList<>();

        for (int i = 0; i < numCourses; i++) {
            if (indegree[i] == 0) {
                q.offer(i);
            }
        }

        // Step 4: BFS — write polled node into order
        int[] order = new int[numCourses];
        int idx = 0;

        while (!q.isEmpty()) {
            int u = q.poll();
            order[idx++] = u;

            for (int v : adj.get(u)) {
                indegree[v]--;

                if (indegree[v] == 0) {
                    q.offer(v);
                }
            }
        }

        // Step 5: if cycle blocked us, return empty
        if (idx != numCourses) {
            return new int[0];
        }

        return order;
    }
}

Compared to 207

Two tiny edits over the boolean version in Blind 75 #45 — 207. Course Schedule:

1. Replace count++ with order[idx++] = u; inside the BFS.
2. Return order when idx == numCourses; otherwise return new int[0] (a cycle leaves at least one node with non-zero indegree, so idx never reaches numCourses).

Same O(V + E) time and O(V + E) space.
