12. Integer to Roman

Build the Roman numeral greedily by taking the largest symbol-value pair possible at each step until the number becomes 0.

Microsoft · Medium — 15 April 2026

Summary

Hint: Keep descending value-symbol arrays (including subtractive pairs like 900=CM, 90=XC, 4=IV). For each value, append the symbol while num is at least that value, subtract, and continue.

Answers

✅ Greedy approach

✅ Solution.java — greedy 
class Solution {
    public String intToRoman(int num) {
        int[] values = {
            1000, 900, 500, 400,
            100, 90, 50, 40,
            10, 9, 5, 4, 1
        };

        String[] symbols = {
            "M", "CM", "D", "CD",
            "C", "XC", "L", "XL",
            "X", "IX", "V", "IV", "I"
        };

        StringBuilder result = new StringBuilder();

        for (int i = 0; i < values.length; i++) {
            while (num >= values[i]) {
                result.append(symbols[i]);
                num -= values[i];
            }
        }

        return result.toString();
    }
}
Greedy with fixed symbol table. Time O(1) in practice (bounded by 13 symbols), space O(1) auxiliary.

Complexity

  • Time: O(1) (bounded symbol set, max value 3999).
  • Space: O(1) auxiliary (excluding output string).
