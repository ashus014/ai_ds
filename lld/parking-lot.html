Parking Lot

Low-Level Design — multi-level parking, gates, spot allocation, and pricing

April 22, 2026

GitHub — full source (LLD/parkingLot)

Summary

Design a multi-level parking lot with entry and exit gates, ticket generation, spot allocation, cost computation, payment handling, and strategy-based spot lookup and pricing.

Parking Lot File Content (Formatted)

parkingLot/ParkingLot.md

Goals

  1. The parking lot should have multiple floors where customers can park their cars.
  2. The parking lot should have multiple entry and exit points.
  3. Customers can collect a parking ticket from the entry points and can pay the parking fee at the exit points on their way out.
  4. Customers can pay the tickets at the automated exit panel or to the parking attendant.
  5. Customers can pay via both cash and credit cards.
  6. Customers should also be able to pay the parking fee at the customer info portal on each floor. If the customer has paid at the info portal, they do not have to pay at the exit.
  7. The system should not allow more vehicles than the maximum capacity of the parking lot. If the parking is full, the system should show a message at the entrance panel and on the parking display board on the ground floor.
  8. Each parking floor will have many parking spots. The system should support multiple types of parking spots such as Compact, Large, Handicapped, Motorcycle, etc.
  9. The parking lot should have some parking spots specified for electric cars. These spots should have an electric panel through which customers can pay and charge their vehicles.
  10. The system should support parking for different types of vehicles like car, truck, van, motorcycle, etc.
  11. Each parking floor should have a display board showing any free parking spot for each spot type.
  12. The system should support a per-hour parking fee model. For example, customers have to pay $4 for the first hour, $3.5 for the second and third hours, and $2.5 for all the remaining hours.

Flow Description

User interacts with:

  1. Entry Gate
  2. Find Parking Spot
  3. Generate Ticket
  4. Update Parking Spot (which updates the Parking Lot)
  5. Exit Gate
  6. Cost Calculation
  7. Payment
  8. Update Parking Spot -> Free (which updates the Parking Lot)

Parking Lot: State is updated by actions at both the Entry and Exit Gates.

code/building/ParkingBuilding.java

package LLD.parkingLot.code.building;

import LLD.parkingLot.code.model.Ticket;
import LLD.parkingLot.code.model.Vehicle;
import LLD.parkingLot.code.spot.ParkingSpot;

import java.util.ArrayList;
import java.util.List;

public class ParkingBuilding {
    private final List<ParkingLevel> levels;

    public ParkingBuilding(List<ParkingLevel> levels) {
        if (levels == null || levels.isEmpty()) {
            throw new IllegalArgumentException("Levels cannot be null or empty");
        }
        this.levels = new ArrayList<>(levels);
    }

    public Ticket allocate(Vehicle vehicle) {
        if (vehicle == null) {
            throw new IllegalArgumentException("Vehicle cannot be null");
        }

        for (ParkingLevel level : levels) {
            ParkingSpot allocatedSpot = level.park(vehicle.getVehicleType());
            if (allocatedSpot != null) {
                return new Ticket(vehicle, level, allocatedSpot);
            }
        }

        throw new IllegalStateException("Parking is full for vehicle type: " + vehicle.getVehicleType());
    }

    public void release(Ticket ticket) {
        if (ticket == null) {
            throw new IllegalArgumentException("Ticket cannot be null");
        }
        ticket.getLevel().unPark(ticket.getVehicle().getVehicleType(), ticket.getSpot());
    }
}

What this file does: Coordinates parking levels to allocate and release spots for incoming and outgoing vehicles.

code/building/ParkingLevel.java

package LLD.parkingLot.code.building;

import LLD.parkingLot.code.enums.VehicleType;
import LLD.parkingLot.code.manager.ParkingSpotManager;
import LLD.parkingLot.code.spot.ParkingSpot;

import java.util.EnumMap;
import java.util.Map;

public class ParkingLevel {
    private final int levelNumber;
    private final Map<VehicleType, ParkingSpotManager> managers;

    public ParkingLevel(int levelNumber, Map<VehicleType, ParkingSpotManager> managers) {
        if (levelNumber <= 0) {
            throw new IllegalArgumentException("Level number must be positive");
        }
        if (managers == null || managers.isEmpty()) {
            throw new IllegalArgumentException("Managers cannot be null or empty");
        }
        this.levelNumber = levelNumber;
        this.managers = new EnumMap<>(managers);
    }

    public int getLevelNumber() {
        return levelNumber;
    }

    public boolean hasAvailability() {
        return managers.values().stream().anyMatch(ParkingSpotManager::hasFreeSpot);
    }

    public ParkingSpot park(VehicleType type) {
        if (type == null) {
            throw new IllegalArgumentException("Vehicle type cannot be null");
        }

        ParkingSpotManager manager = managers.get(type);
        if (manager == null) {
            return null;
        }
        return manager.park();
    }

    public void unPark(VehicleType type, ParkingSpot spot) {
        if (type == null) {
            throw new IllegalArgumentException("Vehicle type cannot be null");
        }

        ParkingSpotManager manager = managers.get(type);
        if (manager == null) {
            throw new IllegalArgumentException("No manager registered for vehicle type: " + type);
        }
        manager.unPark(spot);
    }
}

What this file does: Represents one parking floor and delegates parking operations to the right spot manager by vehicle type.

    public static void main(String[] args) throws InterruptedException {
        ParkingLot parkingLot = buildParkingLot();

        Vehicle bike1 = new Vehicle("MH12AA1001", VehicleType.TWO_WHEELER);
        Vehicle bike2 = new Vehicle("MH12AA1002", VehicleType.TWO_WHEELER);
        Vehicle car1 = new Vehicle("DL01BB2001", VehicleType.FOUR_WHEELER);

        Ticket bike1Ticket = parkingLot.vehicleArrives(bike1);

        System.out.println("Bike2 parked at level " + bike2Ticket.getLevel().getLevelNumber() + " spot " + bike2Ticket.getSpot().getSpotId());

        EntranceGate entranceGate = new EntranceGate();

        return new ParkingLot(building, entranceGate, exitGate);

                new ParkingSpot("L1-TW-02")
        );

        List<ParkingSpot> fourWheelerSpots = Arrays.asList(
                new ParkingSpot("L1-FW-01"),
                new ParkingSpot("L1-FW-02")
        );

        ParkingSpotManager twoWheelerManager = new TwoWheelerParkingSpotManager(twoWheelerSpots, strategy);
        ParkingSpotManager fourWheelerManager = new FourWheelerParkingSpotManager(fourWheelerSpots, strategy);

        Map<VehicleType, ParkingSpotManager> managers = new EnumMap<>(VehicleType.class);
        managers.put(VehicleType.TWO_WHEELER, twoWheelerManager);
        managers.put(VehicleType.FOUR_WHEELER, fourWheelerManager);
        return new ParkingLevel(1, managers);
    }

    private static ParkingLevel buildLevel2(ParkingSpotLookupStrategy strategy) {
        List<ParkingSpot> twoWheelerSpots = Arrays.asList(
                new ParkingSpot("L2-TW-01")
        );

        List<ParkingSpot> fourWheelerSpots = Arrays.asList(
                new ParkingSpot("L2-FW-01"),
                new ParkingSpot("L2-FW-02")
        );

        ParkingSpotManager twoWheelerManager = new TwoWheelerParkingSpotManager(twoWheelerSpots, strategy);

        Map<VehicleType, ParkingSpotManager> managers = new EnumMap<>(VehicleType.class);

        managers.put(VehicleType.FOUR_WHEELER, fourWheelerManager);
        return new ParkingLevel(2, managers);
    }
}

package LLD.parkingLot.code.gate;

import LLD.parkingLot.code.building.ParkingBuilding;
import LLD.parkingLot.code.cost.CostComputation;
import LLD.parkingLot.code.model.Payment;
import LLD.parkingLot.code.model.Ticket;

public class ExitGate {
    private final CostComputation costComputation;

    public ExitGate(CostComputation costComputation) {
        if (costComputation == null) {
            throw new IllegalArgumentException("Cost computation cannot be null");
        }
        this.costComputation = costComputation;
    }

    public void exit(ParkingBuilding building, Ticket ticket, Payment payment) {
        if (building == null) {
            throw new IllegalArgumentException("Parking building cannot be null");
        }
        if (ticket == null) {
            throw new IllegalArgumentException("Ticket cannot be null");
        }
        if (payment == null) {
            throw new IllegalArgumentException("Payment cannot be null");
        }

        double totalFee = costComputation.compute(ticket);
        if (!payment.isSuccessful() || payment.getAmount() < totalFee) {
            throw new IllegalArgumentException("Payment failed or insufficient amount. Required fee: " + totalFee);
        }

        building.release(ticket);
    }
}

package LLD.parkingLot.code.manager;

import LLD.parkingLot.code.spot.ParkingSpot;
import LLD.parkingLot.code.strategy.lookup.ParkingSpotLookupStrategy;

import java.util.ArrayList;
import java.util.List;
import java.util.concurrent.locks.ReentrantLock;

public abstract class ParkingSpotManager {
    protected final List<ParkingSpot> spots;
    protected final ParkingSpotLookupStrategy strategy;
    protected final ReentrantLock lock;

    protected ParkingSpotManager(List<ParkingSpot> spots, ParkingSpotLookupStrategy strategy) {
        if (spots == null || spots.isEmpty()) {
            throw new IllegalArgumentException("Spots cannot be null or empty");
        }
        if (strategy == null) {
            throw new IllegalArgumentException("Strategy cannot be null");
        }
        this.spots = new ArrayList<>(spots);
        this.strategy = strategy;
        this.lock = new ReentrantLock(true);
    }

    public ParkingSpot park() {
        lock.lock();
        try {
            ParkingSpot selected = strategy.selectSpot(spots);
            if (selected == null) {
                return null;
            }
            selected.occupySpot();
            return selected;
        } finally {
            lock.unlock();
        }
    }

    public void unPark(ParkingSpot spot) {
        if (spot == null) {
            throw new IllegalArgumentException("Spot cannot be null");
        }

        lock.lock();
        try {
            if (!spots.contains(spot)) {
                throw new IllegalArgumentException("Spot does not belong to this manager: " + spot.getSpotId());
            }
            if (!spot.isSpotFree()) {
                spot.releaseSpot();
            }
        } finally {
            lock.unlock();
        }
    }

    public boolean hasFreeSpot() {
        return spots.stream().anyMatch(ParkingSpot::isSpotFree);
    }
}

What this file does: Base abstraction for parking spot management operations like park, unpark, and availability checks.

package LLD.parkingLot.code.model;

import java.util.Objects;

    private final String vehicleNumber;
    private final VehicleType vehicleType;

    public Vehicle(String vehicleNumber, VehicleType vehicleType) {
        if (vehicleNumber == null || vehicleNumber.trim().isEmpty()) {
            throw new IllegalArgumentException("Vehicle number cannot be null or empty");
        }
        if (vehicleType == null) {
            throw new IllegalArgumentException("Vehicle type cannot be null");
        }
        this.vehicleNumber = vehicleNumber.trim();
        this.vehicleType = vehicleType;
    }

    public String getVehicleNumber() {
        return vehicleNumber;
    }

    public VehicleType getVehicleType() {
        return vehicleType;
    }

    @Override
    public boolean equals(Object obj) {
        if (this == obj) {
            return true;
        }
        if (obj == null || getClass() != obj.getClass()) {
            return false;
        }
        Vehicle vehicle = (Vehicle) obj;
        return Objects.equals(vehicleNumber, vehicle.vehicleNumber) && vehicleType == vehicle.vehicleType;
    }

    @Override
    public int hashCode() {
        return Objects.hash(vehicleNumber, vehicleType);
    }
}

package LLD.parkingLot.code.spot;

public class ParkingSpot {
    private final String spotId;
    private boolean isFree;

    public ParkingSpot(String spotId) {
        if (spotId == null || spotId.trim().isEmpty()) {
            throw new IllegalArgumentException("Spot ID cannot be null or empty");
        }
        this.spotId = spotId;
        this.isFree = true;
    }

    public String getSpotId() {
        return spotId;
    }

    public boolean isSpotFree() {
        return isFree;
    }

    public synchronized void occupySpot() {
        if (!isFree) {
            throw new IllegalStateException("Spot is already occupied: " + spotId);
        }
        this.isFree = false;
    }

    public synchronized void releaseSpot() {
        if (isFree) {
            throw new IllegalStateException("Spot is already free: " + spotId);
        }
        this.isFree = true;
    }

    @Override
    public String toString() {
        return "ParkingSpot{" +
                "spotId='" + spotId + '\'' +
                ", isFree=" + isFree +
                '}';
    }
}

package LLD.parkingLot.code.strategy.lookup;

import LLD.parkingLot.code.spot.ParkingSpot;

import java.util.List;
import java.util.concurrent.ThreadLocalRandom;
import java.util.stream.Collectors;

public class RandomSpotLookupStrategy implements ParkingSpotLookupStrategy {

    @Override
    public ParkingSpot selectSpot(List<ParkingSpot> spots) {
        if (spots == null || spots.isEmpty()) {
            return null;
        }

        List<ParkingSpot> freeSpots = spots.stream()
                .filter(ParkingSpot::isSpotFree)
                .collect(Collectors.toList());

        if (freeSpots.isEmpty()) {
            return null;
        }

        int randomIndex = ThreadLocalRandom.current().nextInt(freeSpots.size());
        return freeSpots.get(randomIndex);
    }
}

package LLD.parkingLot.code.strategy.pricing;

import LLD.parkingLot.code.model.Ticket;

import java.time.Duration;
import java.time.LocalDateTime;

public class HourlyPricingStrategy implements PricingStrategy {
    private static final double FIRST_HOUR_RATE = 4.0;
    private static final double SECOND_AND_THIRD_HOUR_RATE = 3.5;
    private static final double REMAINING_HOUR_RATE = 2.5;

    @Override
    public double calculate(Ticket ticket) {
        if (ticket == null) {
            throw new IllegalArgumentException("Ticket cannot be null");
        }

        LocalDateTime exitTime = LocalDateTime.now();
        long totalMinutes = Duration.between(ticket.getEntryTime(), exitTime).toMinutes();
        long hours = Math.max(1, (totalMinutes + 59) / 60);

        double fee = FIRST_HOUR_RATE;
        if (hours > 1) {
            long secondThirdHours = Math.min(2, hours - 1);
            fee += secondThirdHours * SECOND_AND_THIRD_HOUR_RATE;
        }
        if (hours > 3) {
            fee += (hours - 3) * REMAINING_HOUR_RATE;
        }
        return fee;
    }
}

What this file does: Implements an hourly pricing model for parking tickets.
