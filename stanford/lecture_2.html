Quick Recap (from Lecture 1)

This lecture continues supervised learning: you give the algorithm data with correct answers, and it learns to make predictions.

01

ALVINN — A Real-World Supervised Learning Example

The lecture opens with ALVINN (Autonomous Land Vehicle In a Neural Network), from the late 1980s. A camera on a car captured the road ahead; a learning algorithm learned to steer by watching a human driver. It learned to predict: given this image, what steering angle?

This was an early demonstration of learning to drive via supervised learning: many (image → steering angle) pairs, and the algorithm learns the pattern.

02

Housing Prices — Notation

Running example: predicting Portland house prices from size.

  • x — input / feature (e.g. size in sq ft).
  • y — output / target (e.g. price).
  • One pair (x, y) is a training example; the full set is the training set.
  • n — number of training examples.
  • Superscript (i) means the i-th example: x(i), y(i) — not “to the power i”.
  • h — hypothesis: the learned function mapping x → predicted y.

Goal: learn h so h(x) predicts y well on new houses. If y is continuous, it is regression; if y is a small set of categories, classification.

03

Linear Regression

With two features (size x1, bedrooms x2), approximate y as a linear function:

h(x) = θ0 + θ1x1 + θ2x2

The θj are parameters / weights to learn. θ0 is the intercept. Define x0 = 1 so that:

h(x) = θ0x0 + θ1x1 + θ2x2 = θTx

θTx is the dot product of θ and x (vector form). Question: how do we choose θ?

04

Cost Function J(θ)

We want h(x) close to y on each example. The cost / loss (least squares):

J(θ) = ½ Σi (h(x(i)) − y(i))2

Square errors, sum them, multiply by ½ (convenient for derivatives). Goal: find θ that minimizes J(θ).

05

Gradient Descent

Start with an initial θ; repeatedly move θ in the direction that decreases J.

θj := θj − α · (∂J/∂θj)
  • := means update (assignment).
  • α — learning rate (step size).
  • ∂J/∂θj — partial derivative; move opposite to the gradient.

For a single example, the update becomes the LMS (Least Mean Squares) / Widrow–Hoff rule:

θj := θj + α · (y(i) − h(x(i))) · xj(i)

Intuition: update scales with error (y − h). Small error → small change; large error → larger correction.

Each step moves θ toward a minimum. Contour lines of J behave like a topographic map: you step downhill along the surface.

06

Batch vs. Stochastic Gradient Descent

Batch gradient descent

Use all n examples before each update.

Repeat until convergence:
    θⱼ := θⱼ + α · Σᵢ (y⁽ⁱ⁾ − h(x⁽ⁱ⁾)) · xⱼ⁽ⁱ⁾

Pros: smooth; for linear regression, J is convex → global minimum (with suitable α). Cons: expensive per step when n is huge.

Stochastic gradient descent (SGD)

Update after each single example.

Loop:
    for i = 1 to n:
        θⱼ := θⱼ + α · (y⁽ⁱ⁾ − h(x⁽ⁱ⁾)) · xⱼ⁽ⁱ⁾

Pros: fast progress on large data. Cons: oscillates near the minimum rather than settling exactly — usually acceptable in practice.

For linear regression, J(θ) is a convex quadratic (one global minimum); batch GD finds it if α is not too large.

07

Normal Equations (Closed Form)

Instead of iterating, set the gradient of J to zero and solve. Stack features into design matrix X (n × d, one row per example) and targets y.

J(θ) = ½ (Xθ − y)T(Xθ − y)

Setting ∇J = 0 gives the normal equations:

XTX θ = XTy  ⇒  θ = (XTX)−1 XTy

When to use what: normal equations are fine for smaller problems; gradient descent scales better when n or d is very large (inverting XTX is costly).

08

Overfitting & Underfitting (preview)

  • Underfitting — model too simple; misses real structure.
  • Overfitting — model too complex; fits training noise, poor generalization.

The course returns to this with learning theory in more depth.

09

Locally Weighted Linear Regression (LWR)

Standard linear regression fits one global line. LWR reweights training points by distance to the query x: nearby points count more.

Minimize  Σi w(i) (y(i) − θTx(i))2

Example weights (Gaussian kernel):

w(i) = exp( −||x(i) − x||2 / (2τ2) )

τ (bandwidth) controls how fast weights decay with distance. LWR is non-parametric: you keep all training data for each prediction. Ordinary linear regression is parametric: after learning θ, data can be discarded for prediction.
