2542. Maximum Subsequence Score

LeetCode 75 #51 — Sort index pairs by nums2 descending. Sweep: keep a min-heap of size k over nums1 values in the window so the sum is maximal for the current minimum nums2 in the chosen set. Score = (sum of k largest nums1 in the prefix) × (current nums2).

LeetCode 75 / Heap — Priority Queue

Hint

Zip (nums2[i], nums1[i]), sort by nums2 descending. Min-heap stores the k chosen nums1 values; when you advance i, try to swap in a larger nums1 by removing the smallest in the heap. Multiply running sum by the nums2 at the fixed index.

Answers

Both variants are accepted on LeetCode — same idea, different pair representation.

✅ Variant 1 — int[] pairs

Solution.java — int[] pairs 
class Solution {
    public long maxScore(int[] nums1, int[] nums2, int k) {
        int n = nums1.length;
        List<int[]> pairs = new ArrayList<>();

        for(int i = 0; i< n; i++) {
            pairs.add(new int[]{nums2[i], nums1[i]});
        }

        Collections.sort(pairs, (a, b) -> b[0] - a[0]);

        PriorityQueue<Integer> pq = new PriorityQueue<>();
        long kSum = 0;

        for(int i = 0;i <= k - 1; i++) {
            kSum += pairs.get(i)[1];
            pq.offer(pairs.get(i)[1]);
        }

        long result = kSum * pairs.get(k - 1)[0];

        for(int i = k; i < n; i++) {
            kSum += pairs.get(i)[1] - pq.poll();
            pq.offer(pairs.get(i)[1]);

            result = Math.max(result, kSum * pairs.get(i)[0]);
        }

        return result;
    }
}

✅ Variant 2 — Pair class

Solution.java — Pair class 
class Solution {

    class Pair {
        int nums2;
        int nums1;

        Pair(int nums2, int nums1) {
            this.nums2 = nums2;
            this.nums1 = nums1;
        }
    }

    public long maxScore(int[] nums1, int[] nums2, int k) {
        int n = nums1.length;

        List<Pair> pairs = new ArrayList<>();

        for(int i = 0; i < n; i++) {
            pairs.add(new Pair(nums2[i], nums1[i]));
        }

        // sort by nums2 descending
        Collections.sort(pairs, (a, b) -> b.nums2 - a.nums2);

        PriorityQueue<Integer> pq = new PriorityQueue<>();
        long kSum = 0;

        // first k elements
        for(int i = 0; i < k; i++) {
            kSum += pairs.get(i).nums1;
            pq.offer(pairs.get(i).nums1);
        }

        long result = kSum * pairs.get(k - 1).nums2;

        for(int i = k; i < n; i++) {
            kSum += pairs.get(i).nums1 - pq.poll();
            pq.offer(pairs.get(i).nums1);

            result = Math.max(result, kSum * pairs.get(i).nums2);
        }

        return result;
    }
}
