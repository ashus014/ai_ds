334. Increasing Triplet Subsequence

LeetCode 75 #8 — One pass: keep the smallest first and smallest value larger than first as second; any num > second completes an increasing triplet.

LeetCode 75 / Array

Hint

Track the smallest and second smallest; if I find a number greater than both, a triplet exists.

public class Solution {
    public boolean increasingTriplet(int[] nums) {
        // Edge case: If the array has less than 3 elements, return false
        if (nums.length < 3) return false;

        // Initialize first and second to maximum integer values
        int first = Integer.MAX_VALUE, second = Integer.MAX_VALUE;

        // Iterate through the array
        for (int num : nums) {
            if (num <= first) {
                // Update 'first' if the current number is smaller
                first = num;
            } else if (num <= second) {
                // Update 'second' if the current number is larger than 'first' but smaller than 'second'
                second = num;
            } else {
                // If we find a number larger than 'second', we found a triplet
                return true;
            }
        }

        // No triplet found
        return false;
    }
}
