994. Rotting Oranges

LeetCode 75 #48 — Multi-source BFS: enqueue every rotten orange (2), count fresh (1). Each level of BFS rots adjacent fresh cells, decrements fresh, and pushes new rotten cells. Stop when the queue is empty or no fresh remain. Minutes = levels processed; return -1 if any fresh is left.

LeetCode 75 / Graphs — BFS

Hint

Count fresh and seed Queue with all 2 cells. while (!q.isEmpty() && fresh > 0): process size nodes per minute, 4-neighbor spread, set cell to 2. Answer fresh == 0 ? time : -1.

class Solution {
    public int orangesRotting(int[][] grid) {
        Queue<int[]> q = new LinkedList<>();
        int time = 0;
        int fresh = 0;

        int ROWS = grid.length;
        int COLS = grid[0].length;

        // Count fresh oranges and add rotten oranges to the queue
        for (int r = 0; r < ROWS; r++) {
            for (int c = 0; c < COLS; c++) {
                if (grid[r][c] == 1) {
                    fresh++;
                } else if (grid[r][c] == 2) {
                    q.offer(new int[]{r, c});
                }
            }
        }

        // Define the four directions
        int[][] directions = {{0, 1}, {0, -1}, {1, 0}, {-1, 0}};

        // BFS to rot oranges
        while (!q.isEmpty() && fresh > 0) {
            int size = q.size();

            // Process all rotten oranges at the current time
            for (int i = 0; i < size; i++) {

                int[] pos = q.poll();

                // Check all four directions
                for (int[] dir : directions) {
                    int row = pos[0] + dir[0];
                    int col = pos[1] + dir[1];

                    // If out of bound or not fresh, continue
                    if (row < 0 || row == ROWS || col < 0 || col == COLS || grid[row][col] != 1) {
                        continue;
                    }

                    // Now, make the oranges rotten
                    grid[row][col] = 2;
                    q.offer(new int[]{row, col});
                    fresh--;
                }
            }
            // increment the time
            time++;
        }

        return fresh == 0 ? time : -1;
    }
}
