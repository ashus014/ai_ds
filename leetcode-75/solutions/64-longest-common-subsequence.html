1143. Longest Common Subsequence

LeetCode 75 #64 — Given two strings, return the length of their longest common subsequence. From indices (i, j): if text1.charAt(i) == text2.charAt(j), extend with 1 + solve(i+1, j+1); else take max(solve(i+1,j), solve(i,j+1)).

LeetCode 75 / DP — Multidimensional

Hint

End of either string → LCS length 0. Only O(|text1|·|text2|) distinct (i,j) states—memoize.

Answers

Naive recursion

Try match / skip on each axis without storing subproblems.

Naive — reference only 
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        return solve(text1, text2, 0, 0);
    }

    private int solve(String s1, String s2, int i, int j) {
        if(i >= s1.length() || j >= s2.length()) {
            return 0;
        }

        if(s1.charAt(i) == s2.charAt(j)) {
            return 1 + solve(s1, s2, i + 1, j + 1);
        }

        int skipS1 = solve(s1, s2, i + 1, j);
        int skipS2 = solve(s1, s2, i, j + 1);

        return Math.max(skipS1, skipS2);
    }
}
Naive approach. Subproblem (i, j) is recomputed from many different recursion paths, so time is exponential in the string lengths. A 2D memo (or bottom-up DP table) fixes that.

✅ Memoization (top-down DP)

Store the LCS length for each suffix pair (i, j) in memo[i][j].

Solution.java — memoization 
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int [][] memo = new int[text1.length()][text2.length()];
        for(int[] arr : memo) {
            Arrays.fill(arr, -1);
        }
        return solve(text1, text2, 0, 0, memo);
    }

    private int solve(String s1, String s2, int i, int j, int[][] memo) {
        if(i >= s1.length() || j >= s2.length()) {
            return 0;
        }

        if(memo[i][j] != -1) {
            return memo[i][j];
        }

        if(s1.charAt(i) == s2.charAt(j)) {
            memo[i][j] = 1 + solve(s1, s2, i + 1, j + 1, memo);
        } else {
            memo[i][j] = Math.max(solve(s1, s2, i + 1, j, memo), solve(s1, s2, i, j + 1, memo));
        }

        return memo[i][j];
    }
}

Tabulation (bottom-up)

dp[i][j] is the LCS length of prefixes text1[0 … i−1] and text2[0 … j−1]. Extra row/column of zeros handles empty prefixes. Same transitions as recursion, filled in increasing i, j; answer is dp[m][n].

Solution.java — tabulation 
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int m = text1.length();
        int n = text2.length();
        
        // DP table: dp[i][j] represents the LCS length for text1[0...i-1] and text2[0...j-1]
        int[][] dp = new int[m + 1][n + 1];
        
        // Fill the DP table
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    dp[i][j] = 1 + dp[i - 1][j - 1];
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }
        
        return dp[m][n];
    }
}
