151. Reverse Words in a String

LeetCode 75 #6 — trim, split into words, walk from the end with StringBuilder, skip empty chunks from multi-space splits, then trim the result.

LeetCode 75 / Array · String

Hint

Trim spaces, split using regex to avoid empty tokens, then reverse words.

class Solution {
    public String reverseWords(String s) {
        // trim to remove the start and end spaces
        String[] words = s.trim().split(" ");

        StringBuilder sb = new StringBuilder();
        for (int i = words.length - 1; i >= 0; i--) {
            if (words[i].trim().isEmpty() == false) {
                sb.append(words[i]);
                sb.append(" ");
            }
        }
        return sb.toString().trim();
    }
}
